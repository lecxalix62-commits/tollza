# VK Comment Assistant

MVP-проект на `FastAPI` для безопасной автоматизации подготовки комментариев:

- мониторинг найденных постов;
- таргет-фильтрация по профилю;
- генерация черновиков комментариев;
- очередь на ручное подтверждение.
- встроенная веб-панель оператора.

Проект намеренно не содержит модуль автоматической публикации комментариев.

## Что внутри

- `GET /posts` - список постов из mock-источника
- `GET /profiles` - список профилей таргетинга
- `POST /profiles` - создать новый профиль таргетинга
- `GET /posts/matches/{profile_id}` - получить подходящие посты по профилю
- `POST /drafts/generate` - сгенерировать черновик комментария
- `GET /drafts` - посмотреть очередь черновиков
- `POST /drafts/{draft_id}/approve` - одобрить черновик
- `POST /drafts/{draft_id}/reject` - отклонить черновик
- `GET /` - веб-панель оператора

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## Desktop UI на Python

Нативный интерфейс без браузера запускается так:

```bash
python3 -m app.desktop
```

В desktop-режиме приложение использует текущие Python-модели и `SQLite` хранилище напрямую, без HTTP-слоя.

В desktop UI есть:

- светлая тема;
- пакетная подготовка черновиков по найденным совпадениям с последующей ручной модерацией;
- экспорт черновиков в `CSV` и `Excel`;
- открытие ссылки на выбранный пост прямо из приложения;
- фильтры и автогруппировка по сообществам;
- постоянное хранение данных в `SQLite` (`data/vk_comment_assistant.db`).

После запуска:

- веб-панель: `http://127.0.0.1:8000/`
- Swagger UI: `http://127.0.0.1:8000/docs`
- OpenAPI JSON: `http://127.0.0.1:8000/openapi.json`

## Пример сценария

1. Получить профили через `GET /profiles`
2. Проверить совпадения через `GET /posts/matches/{profile_id}`
3. Создать черновик через `POST /drafts/generate`
4. Отдать черновик на ручную модерацию
5. Зафиксировать решение через `approve` или `reject`

## Пример запроса на создание черновика

```json
{
  "post_id": "post-1",
  "profile_id": "profile-REPLACE_ME",
  "tone": "expert",

  "call_to_action": "Если нужно, могу показать, какие поля и отчеты обычно настраивают первыми"

```
