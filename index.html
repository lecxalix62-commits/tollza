Вопросы

Часто спрашивают

Как работает вход через ВКонтакте?

Кнопка «Войти через ВК» открывает официальную OAuth-страницу ВКонтакте. ТУЛЗА получает access_token со скоупами wall, photos, messages, offline — этого достаточно для публикаций, фото и ЛС. Логин и пароль никуда не передаются.

А не забанит ли ВК за автокомментинг?

Каждое сообщение можно отправить на ручную модерацию через очередь черновиков. Автопилот соблюдает интервалы между постами и лимит на сообщество. Никакого спама — только подтверждённые вами сообщения.

Что значит «AI-генерация изображений»?

На тарифе «Про» подключаются движки NanoBanana и GPT-Image. Берём картинку из чужого поста и трансформируем её в уникальный визуал — без копирайтных рисков.

Есть ли API?

Да. Полный OpenAPI и Swagger UI доступны на /docs после входа. На тарифе «Команда» вы получаете токен для интеграций. Все методы перечислены в секции «Инструкция».

Можно ли запустить локально?

Да. Это FastAPI-приложение, исходники в папке vk-comment-assistant/. Команды для запуска — в разделе «Инструкция».

Быстрый старт

  1. Жмёте «Войти через ВКонтакте». ТУЛЗА откроет страницу oauth.vk.com/authorize со скоупами wall, photos, messages, offline.
  2. Подтверждаете доступ. VK вернёт токен — мы сохраним его в .env вашего рабочего пространства и сразу откроем панель.
  3. Добавляете сообщества. В разделе «Сообщества» вставьте ссылку vk.com/club… или найдите через встроенный поиск.
  4. Настраиваете автопилот. Укажите ключевые слова, текст сообщения и интервал. Бот заработает в фоне.
  5. Модерируете очередь. Подтверждайте или отклоняйте каждое сообщение в очереди черновиков — спам исключён.

🖥 Локально

FastAPI на своём сервере

Исходники лежат в vk-comment-assistant/. Версия 0.4.0 на FastAPI, хранилище SQLite, опциональный desktop-режим без браузера.

# 1. установка зависимостей
cd vk-comment-assistant
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 2. запуск веб-панели
uvicorn app.main:app --reload
# → http://127.0.0.1:8000/

# 3. (опционально) desktop-режим без браузера
python3 -m app.desktop
Swagger UI OpenAPI JSON /health

Основные API-маршруты

GET /auth/vk Редирект на OAuth ВКонтакте — получаем access_token
GET /auth/status Текущее состояние подключения и превью токена
GET /communities Список ваших сообществ ВК в работе
POST /communities/resolve Добавить сообщество по ссылке vk.com/club…
GET /communities/search-vk?q=... Поиск сообществ по фразе
POST /drafts/generate Сгенерировать черновик комментария под пост
GET /drafts Очередь черновиков на ручную модерацию
POST /drafts/{id}/approve Одобрить и опубликовать черновик
POST /drafts/{id}/reject Отклонить черновик
GET /autopilot/config Текущие настройки автопилота
POST /autopilot/config Сохранить ключи, шаблон сообщения, интервал
GET /monitor/matches/{profile_id} Подходящие посты по профилю таргетинга
