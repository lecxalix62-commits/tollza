# ТУЛЗА — лендинг VK Comment Assistant

Презентационный сайт + ссылки входа в продукт **VK Comment Assistant**
(`vk-comment-assistant/` — FastAPI 0.4.0).

## Структура

- `index.html`, `styles.css`, `main.js` — лендинг (моховой дизайн, мини-игра, мокапы, инструкция, документация API)
- `vk-comment-assistant/` — сам продукт: FastAPI-приложение для безопасной автоматизации ВКонтакте

## Как запустить вместе

Сайт-лендинг рассчитан на то, что **FastAPI-продукт обслуживает корень**, а статика лендинга
монтируется как замена для `/`. Самый короткий путь:

```bash
cd vk-comment-assistant
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# Опционально: подменить landing на наш мховой сайт
cp ../index.html ../styles.css ../main.js app/static/

uvicorn app.main:app --reload
```

После старта:

| URL | Что отдаёт |
|---|---|
| `http://127.0.0.1:8000/` | лендинг (`landing.html`) |
| `http://127.0.0.1:8000/auth/vk` | редирект на OAuth ВКонтакте |
| `http://127.0.0.1:8000/auth/callback` | приём токена и перенос в кабинет |
| `http://127.0.0.1:8000/app` | панель управления |
| `http://127.0.0.1:8000/docs` | Swagger UI |
| `http://127.0.0.1:8000/health` | проверка живости |

## Что есть на сайте

- **Hero** — мховый фон в три параллакс-слоя + переход «Войти через ВКонтакте» (`/auth/vk`)
- **Возможности** — 6 карточек, отражающих реальный продукт: массовые комментарии, ЛС, мониторинг, автопилот 24/7, AI-генерация (NanoBanana + GPT-Image), очередь модерации
- **Интерфейс** — три мокапа экранов: аналитика, переписка, парсер
- **Как работает** — 4 шага в темпе реального продукта: вход → сообщества → автопилот → лиды
- **Мини-игра** — «Поймай комментарий», canvas, рекорд в localStorage
- **Тарифы** — `Старт 500 ₽`, `Про 1 490 ₽`, `Команда 3 990 ₽` (реальные тарифы продукта)
- **Инструкция** — облачный быстрый старт + локальная установка + таблица API-маршрутов + пример `POST /drafts/generate`
- **FAQ** — пять вопросов про OAuth, баны, AI, API и локальный запуск
- **Отзывы** — три карточки с dicebear-аватарами
- **CTA** — кнопка «Войти через ВКонтакте» вместо email-формы

## Вход в продукт

С лендинга в продукт ведут шесть точек входа:

1. `nav` → кнопка «Войти через ВК» → `/auth/vk`
2. `hero` → primary CTA → `/auth/vk`
3. `hero` → ghost CTA «Открыть панель» → `/app`
4. `pricing` → кнопки тарифов → `/auth/vk`
5. `instruction` → кнопки колонки «В облаке» → `/auth/vk` и `/app`
6. `footer` → ссылки на `/auth/vk`, `/app`, `/docs`, `#instruction`

Кнопка «Войти через ВК» открывает официальную страницу OAuth ВКонтакте со скоупами
`wall, photos, messages, offline`. Логин и пароль пользователя ТУЛЗА не видит.
