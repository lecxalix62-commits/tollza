# ТУЛЗА

Презентационный сайт сервиса **ТУЛЗА** — автокомментинг и авторассылка сообщений в ВКонтакте.

## Запуск

Откройте `index.html` в браузере. Никаких сборок не требуется — чистые HTML/CSS/JS.

```bash
# Или поднимите локальный сервер
python3 -m http.server 8000
# и откройте http://localhost:8000
```

## Из чего сделано

- `index.html` — структура страницы (hero, возможности, шаги, тарифы, FAQ, CTA)
- `styles.css` — палитра «глубокий лес + светящийся мох», адаптив, анимации
- `main.js` — споры, параллакс, печатающийся терминал, счётчики, колыхание травы

## Особенности

- Реалистичный мох на переднем плане через SVG-турбуленцию и паттерны
- Папоротники, грибы, капли росы, плавающие споры
- Параллакс скролла + параллакс курсора на превью
- Покадровый терминал «запуск ТУЛЗЫ»
- Счётчики с easing, реактивные шаги, мерцающая роса
